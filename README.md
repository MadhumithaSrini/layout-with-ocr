# layout-with-ocr
To run lp_ocr.py: 
Wrapper for layout parser OCR. 
- Create a venv and activate:  
1. virtualenv lp_ocr
2. source lp_ocr/bin/activate
- Install all packages in the environment: 
1. pip3 install -r requirements.txt
2. apt install < requirements.txt
- Run lp_ocr script: 
1. python3 lp_ocr.py
