# layout-with-ocr
To run lp_ocr.py: 
Wrapper for layout parser OCR. 
1. Create a venv and activate - 
virtualenv lp_ocr
source lp_ocr/bin/activate
2. Install all packages in the environment. 
pip install -r requirements.txt
apt install < requirements.txt
3. Run lp_ocr script 
python3 lp_ocr.py
